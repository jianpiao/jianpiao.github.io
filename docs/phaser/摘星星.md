# Phaser游戏引擎初体验

## 前言

游戏引擎除了coco2d，unity3d，egret，其实还有一些在国内不太知名的框架，一样是可以写出好玩的游戏，这里介绍一下Phaser。

## Phaser是什么？

Phaser是一个HTML5游戏框架，专注于桌面和移动端的HTML5游戏引擎，它的游戏动画需要用到canvas。

有尝试过coco2d，unity3d，egret等等游戏引擎，它们的特点是具有丰富的IDE、工具链，缺点是上手成本太高，初学者配置一个开发环节就要半天。Phaser只需要引入CDN链接就可以正常开发了，上手和学习成本极低。对于开发所使用的IDE也没有限制，完全可以使用自己最擅长的vscode。

## 摘星星

这里就写一个摘星星游戏来体验一下phaser引擎做游戏开发，使用的版本是phaser3。

**游戏规则：**

* 有三排星星，星星的数量每排24个
* 可以计分，摘一颗星星记10分
* 星星摘完之后能够实现下一轮游戏，计分累加
* 从第二轮开始，每新增一轮，出现一枚炸弹，摘星星时不能碰撞到炸弹，否则游戏结束
* 游戏结束后能够重新开始

### 游戏环境搭建

第一步是得搭建一个游戏开发的环境，我这里创建一个html文件，先实例一个`game`，做好基础的配置。

最后将配置项放入在`Phaser.Game`中创建一个游戏画布。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <script src="https://cdn.jsdelivr.net/npm/phaser@3.11.0/dist/phaser.js"></script>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>phaser游戏</title>
</head>
<style>
  body {
    margin: 0;
  }
</style>

<body>
  <script>
    var config = {
      // [CANVAS | WEBGL | AUTO] 推荐AUTO
      // 它将自动尝试使用WebGL，如果浏览器或设备不支持，它将回退为Canvas。
      type: Phaser.AUTO,
      width: 800, // 游戏区域宽度
      height: 600, // 游戏区域高度
      physics: {
        default: 'arcade', // 使用arcade物理引擎
        arcade: {
          gravity: {
            y: 300 // 物体的重力
          },
          debug: false // 关闭debug模式
        }
      },
      scene: { // 场景
        preload: preload, // 提前加载物料
        create: create, // 创建游戏视图层
        update: update // 更新游戏动画
      }
    };

		// 创建一个游戏，实例之后会在window中注入Phaser的原型属性和方法
    // 之后，通过this即可访问phaser的属性和方法
		var game = new Phaser.Game(config);

    function preload() {}

    function create() {}

    function update() {}
  </script>
</body>

</html>
```

### 加载资源

画布创建好之后，需要给画布加载所需要的资源。所有的资源都需要通过`preload`预加载函数内部，调用Phaser的Loader加载器。当然，我们不需要手动去调用Loader加载器，Phaser启动的时候回自动找到这个函数，并且加载里面所有定义好的资源。

在`preload`函数中添加一点资源，如下：

```javascript
function preload() {
  this.load.image('sky', 'assets/sky.png');
  this.load.image('star', 'assets/star.png');
  this.load.image('bomb', 'assets/bomb.png');
  this.load.image('ground', 'assets/platform.png');
  this.load.spritesheet('dude',
    'assets/dude.png', {
    frameWidth: 32,
    frameHeight: 48
  }
 );
}
```

总共加载了5个资源，分别是

* sky背景图
* star星星
* bomb炸弹
* ground站台
* dude玩家

这里要额外解释一下，dude玩家使用的`spritesheet`精灵表单，它是整个游戏引擎的核心物件，它具有动态物理属性，比如移动位置，物体变化，碰撞，反弹等等，而`images`一般是静态物体，只具备一些基本的物理，比如碰撞，反弹等等。

玩家点按左边，要让人物向左移动，点按右边要让人物向右移动，点按上要让人物跳起来，点按下则让人物掉下来。所以就有了下面的代码

```javascript
function update() {
  if (gameOver) {
    return;
  }

  if (cursors.left.isDown) {
    player.setVelocityX(-220); // 向左移动的速度

    player.anims.play('left', true);
  } else if (cursors.right.isDown) {
    player.setVelocityX(220); // 向右移动的速度

    player.anims.play('right', true);
  } else {
    player.setVelocityX(0);

    player.anims.play('turn'); // 向上
  }

  if (cursors.up.isDown && player.body.touching.down) {
    player.setVelocityY(-520); // 上升和下落的速度
  }
}
```

由于人物的移动会引发游戏画面的更新，也就是说人物移动一下，画布就得切换一帧。因此点按键盘事件需要放置在`update`函数中。

#### 创造星星

星星要防止在游戏画布上面，游戏画布的x轴和y轴定位是以(0,0)开始算起的。

这里创造三排星星，每排y轴间隔25px，x轴间隔10px，做一个梯形的画布图层。

```javascript
stars = this.physics.add.staticGroup();
// 添加物理区域
for (let i = 0; i <= 24; i++) {
  stars.create(30 + (i * 30), 60, 'star') // x轴和y轴
}
for (let i = 0; i <= 24; i++) {
  stars.create(40 + (i * 30), 85, 'star')
}
for (let i = 0; i <= 24; i++) {
  stars.create(50 + (i * 30), 110, 'star')
}
```

#### 炸弹和计分

为了给游戏一点难度，添加一个炸弹，从游戏的第二轮开始，每一轮成功后都加多一个炸弹，直接游戏结束为止。

为了游戏的可玩性，人物每摘一颗星星就加10分。

```javascript
// 创建一个炸弹，由于不止一个炸弹，所以使用了组模式
bombs = this.physics.add.group();

//  分数
scoreText = this.add.text(16, 16, '分数: 0', {
  fontSize: '22px', // 字体大小
  fill: '#fff' // 字体颜色
});
```

#### 碰撞

前面给每个物件都添加了物理的属性，但是物理之间是存在碰撞的，那么得提前设置好，谁和谁碰撞会产生物理效果。

```javascript
//  将人物和星星与平台进行碰撞
this.physics.add.collider(player, platforms);
this.physics.add.collider(stars, platforms);
this.physics.add.collider(bombs, platforms);

//  检查人物是否与任何星星重叠，如果他调用了collectStar函数
this.physics.add.overlap(player, stars, collectStar, null, this);
// 人物和炸弹碰撞，碰撞之后暂停游戏
this.physics.add.collider(player, bombs, hitBomb, null, this);
```

#### collectStar函数

负责收集星星。

```javascript
function collectStar(player, star) {
  star.disableBody(true, true);

  //  分数累加，并且更新分数
  score += 10;
  scoreText.setText('分数: ' + score);

  if (stars.countActive(true) === 0) {
    setTimeout(() => {
      //  新一轮星星
      stars.children.iterate(function (child) {

        child.enableBody(true, child.x, 0, true, true); // 设置星星有物理属性

      });
    }, 1500);

    // 根据人物的位置，合理的弹出炸弹
    var x = (player.x < 400) ? Phaser.Math.Between(400, 800) : Phaser.Math.Between(0, 400);
    // 随机分配个位置出现炸弹
    var bomb = bombs.create(x, 16, 'bomb');
    bomb.setBounce(1); // 设置炸弹弹性为1，保持恒定的弹性
    bomb.setCollideWorldBounds(true); // 和平台碰撞
    bomb.setVelocity(Phaser.Math.Between(-200, 200), 20); // 设置x和y轴的速度
    bomb.allowGravity = false; // 没有重力

  }
}
```

#### hitBomb函数

当任务碰撞到炸弹之后结束游戏。

```javascript
// 人物碰撞到炸弹
function hitBomb(player, bomb) {
  this.physics.pause(); // 物理引擎暂停

  player.setTint(0xff0000); /// 改变人物的颜色

  player.anims.play('turn'); // 设置动画为第4帧

  gameOver = true; // 游戏结束
  
  reGameBtn.setVisible(true) // 显示重来按钮
  reGameBtnText.setVisible(true)
}
```

#### 游戏重来

游戏结束之后，可以重新开始。所以添加一个按钮和文字，放置在`create`函数中。

```javascript
function create(){
  ....
  createReGameBtn.call(this)
  ...
}
```

`create`函数如下

```javascript
function createReGameBtn() {
  // 创建一个按钮
  reGameBtn = this.add.tileSprite(this.cameras.main.centerX, this.cameras.main.centerY, 140, 60, 'sky')
    .setInteractive()
    .on('pointerdown', () => {
    reGameBtn.setVisible(false)
    reGameBtnText.setVisible(false)

    this.scene.restart();
    gameOver = false
  })

  // 在按钮上添加文字
  reGameBtnText = this.add.text(this.cameras.main.centerX - 44, this.cameras.main.centerY - 8, '重新开始', {
    fontSize: '22px',
    fill: '#fff'
  })
}
```

    let game = new Phaser.Game(config); // 实例化游戏

        key: 'turn',
        frames: [{
          key: 'dude',
          frame: 4

        frameRate: 20
      });

      this.anims.create({
        key: 'right',
        frames: this.anims.generateFrameNumbers('dude', {
          start: 5,
          end: 8
        }),
        frameRate: 10,
        repeat: -1
      });

      cursors = this.input.keyboard.createCursorKeys();

      stars = this.physics.add.staticGroup();

      for (let i = 0; i <= 24; i++) {
        stars.create(30 + (i * 30), 60, 'star')
      }
      for (let i = 0; i <= 24; i++) {
        stars.create(40 + (i * 30), 85, 'star')

      score += 10;
      scoreText.setText('分数: ' + score);

            child.enableBody(true, child.x, 0, true, true); // 设置星星有物理属性

          });
        }, 1500);

        // 根据人物的位置，合理的弹出炸弹
        var x = (player.x < 400) ? Phaser.Math.Between(400, 800) : Phaser.Math.Between(0, 400);
        // 随机分配个位置出现炸弹
        var bomb = bombs.create(x, 16, 'bomb');
        bomb.setBounce(1); // 设置炸弹弹性为1，保持恒定的弹性
        bomb.setCollideWorldBounds(true); // 和平台碰撞
        bomb.setVelocity(Phaser.Math.Between(-200, 200), 20); // 设置x和y轴的速度
        bomb.allowGravity = false; // 没有重力

      }
    }

    // 人物碰撞到炸弹
    function hitBomb(player, bomb) {
      this.physics.pause(); // 物理引擎暂停

      player.setTint(0xff0000); /// 改变人物的颜色

      player.anims.play('turn'); // 设置动画为第4帧

      gameOver = true; // 游戏结束

      reGameBtn.setVisible(true) // 显示重来按钮
      reGameBtnText.setVisible(true)
    }

    function createReGameBtn() {
      // 创建一个按钮
      reGameBtn = this.add.tileSprite(this.cameras.main.centerX, this.cameras.main.centerY, 140, 60, 'sky')
        .setInteractive()
        .on('pointerdown', () => {
          reGameBtn.setVisible(false)
          reGameBtnText.setVisible(false)

          this.scene.restart();
          gameOver = false
        })

      // 在按钮上添加文字
      reGameBtnText = this.add.text(this.cameras.main.centerX - 44, this.cameras.main.centerY - 8, '重新开始', {
        fontSize: '22px',
        fill: '#fff'
      })
    }
  </script>
</body>
